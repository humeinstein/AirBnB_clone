# AirBnB_Clone

A command interpreter to manage objects for Airbnb project at Holberton School.


  - Create Object
  - Get Object
  - Update Object
  - Destroy Object

### Installation

AirBnB requires [python3](https://python.org/) to run.

Clone this repo
```sh
$ cd AirBnb_clone
$ ./console #interactive
$ echo <command> #noninteractive
```
